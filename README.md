# Echo System Info

Echo System Info is a minimal system information viewer that provides
essential hardware and OS details without unnecessary features or tracking.

Built to start fast and stay simple.

## Features
- CPU, RAM, OS and system overview
- Fast startup
- No telemetry
- Minimal user interface
- Low resource usage

## Ideal for
- Quick system checks
- Older or weak hardware
- Privacy-focused environments

## Status
Beta

## Philosophy
Independent software. Privacy by design.
